# ResQlink

## Project Overview
ResQlink is a software engineering project designed to provide emergency assistance and resource coordination during critical situations.

## Features
- Emergency request management
- Resource tracking
- User-friendly interface
- Real-time communication support

## Technologies Used
- Java
- MySQL
- HTML/CSS
- JavaScript


## How to Run
1. Download the project files.
2. Open the project in your IDE.
3. Configure the database.
4. Run the application.

- Mobile application support
